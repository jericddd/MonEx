Connect @monexmonad Live

Follow these steps so real X tweets become catches players can claim in the game.

Step 1 — X Developer Portal

  1. Go to developer.x.com
  2. Sign in as @monexmonad (the bot account)
  3. Create a Project + App
  4. Choose a paid API tier if required (X API is mostly paid in 2026)
  5. App permissions: Read is enough for now (no replies yet)

Step 2 — Get 4 keys

In your app → Keys and tokens → generate OAuth 1.0a:

  • API Key
  • API Secret
  • Access Token
  • Access Token Secret

Important: Generate tokens while logged in as @monexmonad.

Step 3 — Railway Variables

Open Railway → MonEx → Variables → add:

X_API_KEY=your_key
X_API_SECRET=your_secret
X_ACCESS_TOKEN=your_token
X_ACCESS_TOKEN_SECRET=your_token_secret
BOT_USERNAME=monexmonad
ENABLE_X_POLL=1
STARTING_MONBALLS=10

After deploy, set js/monex-config.js with your Worker URL. Use Cloudflare deploy guide (free) instead of Railway.

Step 4 — Test on X

From your personal X account, tweet:

@monexmonad catch 10 monanimals

Wait ~1 minute → check home page X Wild Log.

Step 5 — Claim in game

  1. Open the game → Profile tab
  2. Save your real X @handle (same account you tweeted from)
  3. Click CLAIM TO GAME
  4. Mons appear in Party or Box

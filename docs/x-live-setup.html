Connect @monexmonad Live

Follow these steps so real X tweets become catches players can claim in the game.

Step 1 — X Developer Portal

  1. Go to developer.x.com
  2. Sign in as @monexmonad (the bot account)
  3. Create a Project + App
  4. Choose a paid API tier if required (X API is mostly paid in 2026)
  5. App permissions: Read and Write if you want @monexmonad to reply to catches (ENABLE_X_REPLY=1). Read only is enough for silent catch logging.

Step 2 — Get 4 keys

In your app → Keys and tokens → generate OAuth 1.0a:

  • API Key
  • API Secret
  • Access Token
  • Access Token Secret

Important: Generate tokens while logged in as @monexmonad.

Step 3 — Cloudflare Worker secrets

Cloudflare dashboard → Workers & Pages → monex-api → Settings → Variables → add secrets:

X_API_KEY=your_key
X_API_SECRET=your_secret
X_ACCESS_TOKEN=your_token
X_ACCESS_TOKEN_SECRET=your_token_secret

Set variable ENABLE_X_POLL = 1 (cron polls every 2 min).

Optional: ENABLE_X_REPLY = 1 so @monexmonad replies to catch tweets (requires app Read + Write — see Step 1).

Game API URL: js/monex-config.js — see Cloudflare deploy guide.

Step 4 — Enable replies (optional)

  1. X Developer Portal → your app → User authentication settings → App permissions → Read and write
  2. Regenerate Access Token & Secret while logged in as @monexmonad
  3. Update Cloudflare secrets X_ACCESS_TOKEN and X_ACCESS_TOKEN_SECRET
  4. Set ENABLE_X_REPLY = 1 in Cloudflare Worker variables (or wrangler.toml)
  5. Deploy API (GitHub Actions → Deploy Cloudflare API)

Valid catch examples the bot will reply to:

Reply on a @monexmonad post (no @ needed):
catch
catch 10
catch 10 monanimal
catch 10 monanimals
catch 5 / catch 7 / catch 10  (higher number wins if several appear)

Anywhere else on X (must include @monexmonad):
@monexmonad catch
@monexmonad catch 10 monanimals
@monexmonad catch 10

Step 5 — Test on X

From your personal X account, tweet:

@monexmonad catch 10 monanimals
—or reply on a @monexmonad post with just: catch 10

Wait ~1 minute → check home page X Wild Log.

Step 6 — Sync in game

  1. Open the game → login with your X @handle
  2. Catches auto-sync to Party then Box
  3. Check Profile for your X catch log

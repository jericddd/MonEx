One-time setup (API token — free)

Add Cloudflare Pages permission to the same token you already use for the API deploy.

  1. Cloudflare → API Tokens
  2. Edit your existing deploy token (or create a new one)
  3. Permissions — add:
    Account → Cloudflare Pages → Edit
    Keep: Account → Workers Scripts → Edit
  4. If you created a new token, paste it in GitHub → repo Settings → Secrets → Actions → CLOUDFLARE_API_TOKEN
  5. Push to staging or run Deploy Staging (Cloudflare Pages) in Actions

Staging URL appears in the Actions log after the first successful deploy.

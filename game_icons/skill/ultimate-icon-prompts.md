# MonEx — Ultimate skill icon prompts (17 species)

Use with any image model (Midjourney, DALL·E, Flux, etc.).  
Output: **64×64 or 128×128 PNG**, transparent background, **kebab-case filename** in `game_icons/skill/`.

---

## Shared master block (prepend to every prompt)

```
MonEx ultimate skill icon — retro pixel-art JRPG ability icon for a monster battler.

FORMAT: 64×64 pixels (or 128×128), PNG, transparent background, crisp pixel edges, no blur.

STYLE: Bold single-symbol silhouette, high contrast, limited palette (6–10 colors). Match existing MonEx skill icons (slash, flame-burst, frost-shard): chunky arcade shapes, 1–2px dark outline, readable at 22px UI size.

ULTIMATE TIER (required): Golden amber starburst frame or ring (#fde68a, #f59e0b, #b45309) around the symbol. Inner art uses element colors. More ornate than normal skills — subtle sparks/glow, no text.

RULES: NO letters, NO watermark, NO photorealism, NO 3D render, NO full character portrait. Animal inspiration may appear as a small stylized motif (silhouette, claw, fin, etc.) — not a full body scene.

COMPOSITION: Centered, ~75–85% canvas fill, one clear focal symbol.
```
